## Installation

```shell script
pip install liberty
```

---

>  **Note for Windows!**

1. Delete `PyOpenGL`

    ```shell script
    pip uninstall PyOpenGL
    ```

2. Download and installing [PyOpenGL](https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyopengl)

---

## Update

```shell script
pip install --upgrade liberty
```
